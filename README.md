# StrengthPeaking Calculators

StrengthPeaking is a free collection of strength training and powerlifting calculators for lifters, coaches, and gym users.

The tools are designed to make common training calculations easier, including one-rep max estimates, powerlifting scoring formulas, and percentage-based program calculations.

## Calculators

* [Smolov Jr Calculator](https://strengthpeaking.com/smolov-jr-calculator)
  Build a 3-week Smolov Jr cycle using your chosen lift and estimated max.

* [Bench Press 1RM Calculator](https://strengthpeaking.com/bench-press-1rm-calculator)
  Estimate your bench press one-rep max from reps and weight lifted.

* [DOTS Calculator](https://strengthpeaking.com/dots-calculator)
  Calculate DOTS score for powerlifting performance comparisons.

* [Wilks Calculator](https://strengthpeaking.com/wilks-calculator)
  Calculate Wilks score based on bodyweight and total.

* [5/3/1 Calculator](https://strengthpeaking.com/531-calculator)
  Generate training percentages for Jim Wendler’s 5/3/1 style programming.

* [DUP Program Calculator](https://strengthpeaking.com/dup-program-calculator)
  Create daily undulating periodization training percentages.

* [Wave Loading Calculator](https://strengthpeaking.com/wave-loading-calculator)
  Plan wave loading sets using percentage-based progression.

## About

These calculators are intended as simple training tools. They are not a replacement for coaching, medical advice, or individual programming decisions.

Use sensible loading, adjust based on recovery, and avoid running aggressive programs without appropriate training experience.
